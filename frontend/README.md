CodeBotix is an B2C EdTech platform for middle school and high school students in which they enroll into courses of Robotics, AI and 3D Design. Kids work on components and create projects. The course is split into 3 levels where they can learn on their own pace. It promotes STEM Education by having a structured learning path. AI Concepts are taught in an age appropriate way. Kids understand how machines see, hear and make decisions. It improves the logical thinking and problem solving ability of children. There are visible outcomes from CodeBotix where a child starts as an explorer then becomes a builder in which it eventually leads him to be an innovator. 
# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh
